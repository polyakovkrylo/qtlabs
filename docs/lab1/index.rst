Lab 1. Qt basics
=======================================

This chapter will guide you through corner stones of C++ based application \
development using Qt. We will start with overview of typical main.cpp file for \
Qt project, then proceed with contents of  C++ based Qt basic modules, such as \
QtCore, QtGui, and QtWidgets. Later on we will look at QObject class, \
which is the corner stone of Qt.
